⚙ Kite Connect Settings

Configure TOTP auto-login to sync your Zerodha holdings.
🤖 How TOTP Auto-Login works:
Run kite_auto_login.py once each morning — it logs into Zerodha using your user ID + password + TOTP secret automatically, gets a fresh access token, and saves it to budget.db. Then just click Sync Holdings.
Step 1 — Install dependencies
pip install requests pyotp
Step 2 — Fill in your credentials below, then copy the generated script
⚠ This is the secret key (shown when you scan the QR code during 2FA setup), NOT a TOTP code. Use a QR reader app on the Zerodha 2FA page to extract it.
Generated script — kite_auto_login.py
Step 3 — Run the script each morning before syncing:
python kite_auto_login.py

Or schedule it automatically (Mac crontab):
Run crontab -e and add:
0 8 * * 1-5 cd /path/to/budgetcraft-local && python kite_auto_login.py
(Runs Mon–Fri at 8 AM — after markets open, before your sync)
